# Tailor Template

### clean single-page front-end template built using modern web technologies.

### This project is suitable as a starter template for landing pages and small web projects.

## Features

- Clean and simple layout
- Easy to customize
- Lightweight and fast loading
- Cross-browser compatible

## Built with

- HTML5
- CSS3
- JavaScript (ES6)

## Run

To run the project locally:

1. git clone https://github.com/YoussefAdel3/Tailor-Template.git
2. cd to Mytemp
3. open index.html
